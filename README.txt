NBA Analytics Dashboard - A Power BI project that delves into the 2023/2024 NBA regular season (Excluding the Playoffs), showcasing comprehensive statistics and insights on teams, players, and the league's overall composition. Based on real data, this interactive dashboard allows users to explore key performance indicators, player data, and team comparisons. Access all the necessary files to open in Power BI Desktop and a PDF overview of the project in this repository (note that the PDF does not include interactive features)."
